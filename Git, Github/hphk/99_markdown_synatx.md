# #는 딱 한번만 쓰자.
#나 ##는 '의미가 있는' 애들 앞에 붙이는 거기때문이다. 특히 #는 파일 하나 당 딱 한번만 쓰도록 하자.

## 코드블록
```bash
$ mkdir java-projects  # 폴더 생성(make directory)
$ cd java-projects # 폴더 이동 (change directory)
$ touch a.txt  # 파일 생성
$ ls # 파일/폴더 목록 보기(list)
```
